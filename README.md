# 볼드모트: 어둠의 군주

해리 포터 세계관의 비공식 팬 게임입니다. 플레이어는 볼드모트가 되어 7개의 호크룩스를 지키기 위해 밀려오는 오러와 불사조 기사단의 파상 공세를 막아냅니다. 세로 화면 전용 터치 액션 웨이브 슈터로, 파일 하나(`index.html`)로 동작하며 외부 리소스가 전혀 없습니다.

> 비공식 팬 게임 · Harry Potter는 원작자와 워너브라더스의 자산입니다. 게임 내 모든 스토리와 대사는 창작 텍스트입니다.

## 게임 방법

- **이동**: 화면 아무 곳이나 드래그 (손가락 위치와 캐릭터가 떨어져 있어 시야를 가리지 않습니다)
- **공격**: 자동 — 가장 가까운 적에게 초록 마법 볼트를 발사합니다
- **스킬** (오른쪽 아래 버튼 3개):
  - 아바다 케다브라 — 전방 관통 광선
  - 악령의 불 — 주변을 휩쓰는 화염 고리
  - 호크룩스 결계 — 보호막 + 회복
- 스테이지마다 5개 웨이브 + 보스전. 적이 떨어뜨리는 **영혼 조각**으로 스테이지 사이 '어둠의 제단'에서 공격력·생명력·이동 속도·쿨타임·볼트 개수를 강화하세요.
- 진행 상황(도달 스테이지, 강화, 영혼 조각)은 자동 저장되며 타이틀의 **이어하기**로 계속할 수 있습니다.

## 아이폰에서 플레이하기

1. Safari로 게임 페이지(GitHub Pages 주소 또는 `index.html`)를 엽니다.
2. 첫 터치 시 사운드가 켜집니다. 우측 상단 ♪ 버튼으로 음소거할 수 있습니다.
3. 전체 화면으로 즐기려면 **공유 버튼 → 홈 화면에 추가**를 누른 뒤, 홈 화면 아이콘으로 실행하세요 (주소창 없는 전체 화면 모드로 실행됩니다).
4. 세로(포트레이트) 방향으로 플레이하는 것을 권장합니다.
